## Introduction

Synergy HR - AlignUI HR Management Template for Next.js.

## Getting Started

First, install dependencies:

```bash
pnpm i
```

Run the development server

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
